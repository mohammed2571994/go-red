# mini-mini-go-redis

a simple Redis server implementation, designed for learning Golang and understanding the inner workings of Redis.

## Commands
- [x] PING
- [x] ECHO
- [x] GET 
- [x] SET  
- [x] DEL

- [ ] HGET
- [ ] HGETALL
- [ ] HSET

- [x] INCR
- [x] DECR

## Features
- [ ] AOF Persistence
- [ ] AOF Rewrite
- [ ] Support Concurrent Connections
- [ ] PubSub
- [ ] Replication
